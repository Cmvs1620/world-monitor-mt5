HOW TO PROCEED ON WINDOWS STRATOS
================================================================================

Step 1: Transfer Project Files
  - Copy entire project folder to Windows Stratos
  - Ensure all MQL5 source files are present

Step 2: Compile the EA (if needed)
  - Open MT5 → Tools → MetaEditor (F4)
  - Open file: [project]\ea\GeoSignalEA.mq5
  - Press F5 to compile
  - Verify: "0 error(s)" message appears
  - Compiled .ex5 will appear in MT5 Experts folder

Step 3: Deploy the EA (Option A - Automated)
  - Open PowerShell as Administrator
  - Navigate to project folder
  - Run: .\deploy-ea-to-mt5.ps1 -RestartMT5 $true
  - Script will copy .ex5 and optionally restart MT5

Step 4: Deploy the EA (Option B - Manual)
  - Follow detailed instructions in EA_DEPLOYMENT_GUIDE.md
  - Copy .ex5 file to MT5 Experts folder
  - Restart MT5
  - Open Navigator → Expert Advisors → GeoSignalEA
  - Add to chart

Step 5: Verify Deployment
  - Look for green smiley ✓ in chart corner
  - Check Journal tab (Alt+T) for startup messages
  - Confirm: "[EA] GeoSignal EA Started"

================================================================================
INPUT PARAMETERS ON FIRST DEPLOYMENT
================================================================================

When adding EA to chart, configure (or accept defaults):
  EA_ENABLED = true (enable the EA)
  EA_TRADING_ENABLED = false (demo mode for initial testing)
  POLL_INTERVAL_SECONDS = 60 (checks signal.json every minute)
  SIGNAL_FILE_PATH = "signal.json" (reads from MT5 Files folder)
  DEBUG_MODE = false (set to true for verbose logging)

All other parameters can use defaults initially.
