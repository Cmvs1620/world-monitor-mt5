```python

MIN_CONFIDENCE = 0.50              # Minimum Claude confidence score
```

```mql5
// Expert Advisor properties
```

## Part 2: Signal Data Format Consistency

### Python Signal Format (signal.json)

```json
{
  "timestamp": "2026-04-07T14:23:56.123Z",
  "signals": [
    {
      "event_id": "evt_abc123",
      "symbol": "EURUSD",
      "action": "BUY",
      "confidence": 0.92,
      "reason": "Bullish momentum detected"
    }
  ]
}
```

### EA Parsed Format (internal MQL5 structure)

```c
struct Signal {
  string event_id;         // evt_abc123
  string symbol;           // EURUSD
  string action;           // BUY, SELL, HOLD
  double confidence;       // 0.5-1.0
  string reason;           // Trading reason
};
```

### EA Heartbeat Format (heartbeat.json)

```json
{
  "timestamp": "2026-04-07T14:23:58.456Z",
  "signal_id": "evt_abc123",
  "executed": true,
  "status": "success",
  "ticket": 123456789,
  "order_type": "BUY",
  "symbol": "EURUSD",
  "entry_price": 1.0856,
  "position_size": 0.1,
  "error_reason": ""
}
```

### Consistency Matrix

| Field | Python | EA Reads | EA Writes | Match? |
|-------|--------|----------|-----------|--------|
| event_id / signal_id | ✓ | ✓ | ✓ | Field name differs but data matches |
| symbol | ✓ (EURUSD) | ✓ (EURUSD) | ✓ (EURUSD) | ✓ |
| action | ✓ (BUY/SELL/HOLD) | ✓ (parsed to OP_BUY, OP_SELL) | ✓ (written as "BUY"/"SELL") | ✓ |
| confidence | ✓ (0.92) | ✓ (parsed to double) | ✗ (not in heartbeat) | Partial |
| timestamp | ✓ (ISO 8601) | ✓ (parsed) | ✓ (ISO 8601) | ✓ |
| ticket | N/A in signal.json | ✓ generated by OrderSend | ✓ in heartbeat | ✓ |
| entry_price | N/A | ✓ from OrderSend | ✓ in heartbeat | ✓ |
| position_size | N/A | ✓ calculated | ✓ in heartbeat | ✓ |

Format: `[YYYY-MM-DD HH:MM:SS] LEVEL - message`

```
[2026-04-07 14:23:56] INFO - Signal evt_abc123: BUY EURUSD (confidence: 0.92)
[2026-04-07 14:23:56] INFO - Writing to signals/signal.json
[2026-04-07 14:23:58] INFO - Heartbeat received: signal_id=evt_abc123
[2026-04-07 14:23:58] INFO - Trade executed: ticket=123456789, entry=1.0856
[2026-04-07 14:23:58] INFO - Latency: 2 seconds
```

### Python CSV Logging (logs/signals.csv)

Format: `timestamp,event_id,symbol,action,confidence,ticket,status,latency_sec`

```
2026-04-07T14:23:56.123Z,evt_abc123,EURUSD,BUY,0.92,123456789,success,2
```

### EA Logging (MT5 Journal)

Format: `[HH:MM:SS] GeoSignal EA: message`

```
[14:23:57] GeoSignal EA: Signal received - BUY EURUSD (confidence: 0.92)

[14:23:58] GeoSignal EA: Trade executed - Ticket: 123456789
[14:23:58] GeoSignal EA: Heartbeat written
```
