================================================================================
TROUBLESHOOTING
================================================================================

Issue: signal.json not updating
  → Check Python engine logs: tail -f logs/engine.log
  → Verify API connectivity: curl https://worldmonitor-api/events
  → Restart engine: Ctrl+C, then python run.py

Issue: EA not reading signal
  → Check MT5 Files path in EA properties
  → Verify file permissions in MT5 Files folder
  → Check MT5 Journal for permission errors

Issue: Trade not executing
  → Check MT5 Account balance and margin
  → Review risk gate logs: grep "risk" logs/engine.log
  → Check EA_TRADING_ENABLED flag

Issue: High latency (>5 minutes)
  → Reduce POLL_INTERVAL_SECONDS in run.py (set to 10 for testing)
  → Check network latency: ping stratos.example.com
  → Monitor MT5 disk I/O performance

See TROUBLESHOOTING.md for complete diagnosis guide.
